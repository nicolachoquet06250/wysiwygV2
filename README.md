 - Attributs pris en charge:
    - id
    - style
    - placeholder
    - class
    - resizable
    - resizable-x
    - resizable-y
    - debug
    - spellcheck -> "true" ou "false" ou rien mettre

 - Et le contenu par défault entre les balises ouvrantes et fermantes.
